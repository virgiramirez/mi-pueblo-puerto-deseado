# Mi Pueblo - Puerto Deseado

Proyecto web informativo desarrollado con HTML, CSS y JavaScript.

## Descripción
Este sitio tiene como objetivo centralizar información relevante
para los vecinos y visitantes de Puerto Deseado, incluyendo
datos institucionales, trámites, servicios y turismo.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Navegación entre secciones
- Información institucional de la ciudad
- Acceso a trámites y servicios
- Enlaces externos a sitios oficiales
- Diseño responsive básico

## Objetivo del proyecto
Proyecto realizado con fines educativos como parte de mi formación
en desarrollo web y como primer proyecto para mi repositorio.
